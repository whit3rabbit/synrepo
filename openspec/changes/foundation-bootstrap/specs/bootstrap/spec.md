## ADDED Requirements

### Requirement: Bootstrap must establish contributor guidance
synrepo SHALL define bootstrap guidance that explains how future changes are proposed, named, and sequenced before product feature work expands.

#### Scenario: Prepare the next change after foundation
- **WHEN** a contributor is ready to open the next roadmap-aligned change
- **THEN** the bootstrap guidance identifies the expected naming convention and the next recommended changes
- **AND** the contributor can proceed without redefining the planning workflow
